Here's a breakdown of HTML structure with simple examples for each part:

  1. Doctype Declaration:
    Declares HTML5.

    <!DOCTYPE html>

  2. Root Element (<html>):
    Wraps everything within an HTML document.

    <html lang="en">
           <!-- Content goes here -->
        </html>

  3. Head Section (<head>):
    Contains metadata about the document.

    <head>
           <title>My Web Page</title>
           <meta charset="UTF-8">
           <link rel="stylesheet" href="styles.css">
           <script src="script.js"></script>
        </head>

  4. Body Section (<body>):
    Where visible content lives.

    <body>
           <h1>Welcome to My Website</h1>
           <p>This is a paragraph of text.</p>
        
           <ul>
              <li>Item 1</li>
              <li>Item 2</li>
              <li>Item 3</li>
           </ul>
        
           <a href="https://example.com">Visit Example</a>
        
           <img src="image.jpg" alt="Description of image">
        
           <div>
              <h2>About Us</h2>
              <p>We are dedicated to creating quality content.</p>
           </div>
        </body>

  5. Footer and Header:
    Common for organizing top and bottom sections of the page.

    <header>
           <h1>Site Title</h1>
           <nav>
              <a href="#home">Home</a>
              <a href="#about">About</a>
              <a href="#contact">Contact</a>
           </nav>
        </header>
        
        <footer>
           <p>&copy; 2024 My Website</p>
        </footer>

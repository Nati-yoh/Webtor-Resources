Here’s a guide to HTML forms, covering the <form> tag, various types of <input>, and buttons.

1. Form Tag (<form>)

The <form> tag is used to create an HTML form for collecting user inputs. Key attributes include:

  • action: URL where form data will be sent.
  • method: HTTP method, usually "post" or "get".
<form action="/submit-form" method="post">
   <!-- Form elements go here -->
</form>

2. Text Input (<input type="text">)

Creates a single-line text input field.

<label for="name">Name:</label>
<input type="text" id="name" name="name">

3. Password Input (<input type="password">)

Hides the text as it’s entered, commonly used for passwords.

<label for="password">Password:</label>
<input type="password" id="password" name="password">

4. Email Input (<input type="email">)

Validates the entered text as an email address.

<label for="email">Email:</label>
<input type="email" id="email" name="email">

5. Number Input (<input type="number">)

Accepts only numeric input. Attributes like min, max, and step can control value range.

<label for="age">Age:</label>
<input type="number" id="age" name="age" min="1" max="100">

6. Radio Buttons (<input type="radio">)

Allows users to select one option from a group.

<label>
   <input type="radio" name="gender" value="male"> Male
</label>
<label>
   <input type="radio" name="gender" value="female"> Female
</label>

7. Checkbox (<input type="checkbox">)

Allows users to select multiple options.

<label>
   <input type="checkbox" name="subscribe" value="newsletter"> Subscribe to newsletter
</label>

8. Date Input (<input type="date">)

Provides a date picker for selecting a date.

<label for="birthday">Birthday:</label>
<input type="date" id="birthday" name="birthday">

9. Submit Button (<input type="submit">)

Submits the form data to the server.

<input type="submit" value="Submit">

10. Button Element (<button>)

Provides a clickable button, which can trigger form submission or other actions when combined with JavaScript.

<button type="submit">Submit Form</button>

Full Example

Here’s how these elements look together in a form:

<form action="/submit-form" method="post">
   <label for="name">Name:</label>
   <input type="text" id="name" name="name">

   <label for="email">Email:</label>
   <input type="email" id="email" name="email">

   <label for="password">Password:</label>
   <input type="password" id="password" name="password">

   <label>Gender:</label>
   <input type="radio" name="gender" value="male"> Male
   <input type="radio" name="gender" value="female"> Female

   <label for="age">Age:</label>
   <input type="number" id="age" name="age" min="1" max="100">

   <label for="birthday">Birthday:</label>
   <input type="date" id="birthday" name="birthday">

   <input type="checkbox" name="subscribe" value="newsletter"> Subscribe to newsletter

   <input type="submit" value="Submit">
</form>

This form setup covers basic user input options, ready for collecting data!

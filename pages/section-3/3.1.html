1. color

Controls the text color of an element. This is one of the most common properties used in CSS.

p {
   color: #333; /* Dark gray text */
}

2. background-color

Sets the background color of an element. It can accept color values like color names, hex codes, RGB, and more.

div {
   background-color: lightblue;
}

3. font-size

Defines the size of the font. It can be set in units like pixels (px), ems (em), rems (rem), percentages (%), etc.

h1 {
   font-size: 2rem; /* 2 times the root font size */
}

4. margin

Creates space outside an element, separating it from other elements. You can set margins for all four sides or individually.

p {
   margin: 20px; /* Adds 20px of margin on all sides */
}

5. padding

Adds space inside an element, between its content and border. Like margin, you can define padding for all sides or each side individually.

div {
   padding: 10px 20px; /* 10px for top/bottom, 20px for left/right */
}

6. border

Defines the border around an element, including width, style, and color.

button {
   border: 2px solid #333; /* 2px solid border in dark gray */
}

7. display

Controls the display behavior of an element. Common values are:

  • block: Elements take up the full width of their parent and begin on a new line.
  • inline: Elements take up only as much width as necessary.
  • flex: Used to create flexible layouts.
div {
   display: flex; /* Makes div a flex container */
}

8. position

Specifies how an element is positioned in the document. Common values are:

  • static: The default value, elements are positioned based on the normal document flow.
  • relative: Positions an element relative to its normal position.
  • absolute: Positions an element relative to its nearest positioned ancestor.
  • fixed: Positions an element relative to the browser window.
.fixed-element {
   position: fixed;
   top: 10px;
   right: 10px;
}

9. flexbox (display: flex and related properties)

A powerful layout model that provides more efficient ways to arrange elements in a container. Key properties include justify-content, align-items, and flex-direction.

.container {
   display: flex;
   justify-content: space-between; /* Distributes space between items */
}

.item {
   flex: 1; /* Makes each item grow to take up equal space */
}

Conclusion:

These are fundamental properties that every web designer or developer should understand, as they lay the foundation for most designs. With these, you can control text, layout, and spacing across your web pages effectively.

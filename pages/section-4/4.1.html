Flexbox (Flexible Box Layout) is a CSS layout system that makes it easy to align, distribute, and organize elements within a container, especially when their sizes are dynamic.

What Flexbox Does

  • Aligns items horizontally or vertically without needing complex CSS.
  • Distributes free space between items or pushes them together.
  • Helps create responsive designs by adjusting item size automatically.

How to Apply Flexbox

  1. Enable Flexbox
    Add display: flex; to the parent element.
    Example: 

    .container {
  display: flex;
}

  2. Align Items Along Axes

    • Main Axis: Controlled with flex-direction. It determines the direction of items (row or column).
    • Cross Axis: Controlled with align-items or align-self.

  3. Customize Behavior Use additional properties like justify-content, flex-wrap, and flex (on children) for precise control.

Minimal Example

<div class="container">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

.container {
  display: flex; /* Enables Flexbox */
  justify-content: space-between; /* Spreads items horizontally */
  align-items: center; /* Aligns items vertically */
}

With Flexbox, items adjust to fit their container dynamically, making layouts simple and flexible.

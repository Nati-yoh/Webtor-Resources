Understanding justify-content and align-items in Flexbox

These two properties are the backbone of Flexbox alignment, controlling how items are distributed and aligned along the main axis and the cross axis.

justify-content: Align Along the Main Axis

The justify-content property determines how flex items are spaced along the main axis (horizontal by default).

Values

  1. flex-start (default):
    Items align at the start of the main axis.
    Example: 

    justify-content: flex-start;

  2. flex-end:
    Items align at the end of the main axis.
    Example: 

    justify-content: flex-end;

  3. center:
    Items are centered along the main axis.
    Example: 

    justify-content: center;

  4. space-between:
    Equal space between items, with no space at the edges.
    Example: 

    justify-content: space-between;

  5. space-around:
    Equal space around items, including at the edges (half-sized at the ends).
    Example: 

    justify-content: space-around;

  6. space-evenly:
    Equal space between items, including at the edges.
    Example: 

    justify-content: space-evenly;

align-items: Align Along the Cross Axis

The align-items property controls the alignment of items along the cross axis (vertical by default).

Values

  1. stretch (default):
    Items stretch to fill the container's cross axis.
    Example: 

    align-items: stretch;

  2. flex-start:
    Items align at the start of the cross axis.
    Example: 

    align-items: flex-start;

  3. flex-end:
    Items align at the end of the cross axis.
    Example: 

    align-items: flex-end;

  4. center:
    Items are centered along the cross axis.
    Example: 

    align-items: center;

  5. baseline:
    Items align based on their text baselines.
    Example: 

    align-items: baseline;

Examples

Horizontal Alignment (justify-content)

<div class="container">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

.container {
  display: flex;
  justify-content: space-between; /* Spreads items evenly */
}

Vertical Alignment (align-items)

<div class="container">
  <div>Item 1</div>
  <div style="height: 50px;">Item 2</div>
  <div>Item 3</div>
</div>

.container {
  display: flex;
  align-items: center; /* Aligns items to the center of the container's height */
}

Combining Both

You can use justify-content and align-items together for precise layout control.

<div class="container">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

.container {
  display: flex;
  justify-content: center; /* Center horizontally */
  align-items: flex-start; /* Align items at the top */
  height: 200px; /* To demonstrate vertical alignment */
}

Key Points:

  • justify-content aligns items along the main axis (left-right in row or top-bottom in column).
  • align-items aligns items along the cross axis (top-bottom in row or left-right in column).

They work together to provide full control over Flexbox layouts.

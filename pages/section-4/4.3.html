Flex Properties

The flex property is a shorthand that controls how flex items grow, shrink, and take up space inside a flex container. It consists of three sub-properties: flex-grow, flex-shrink, and flex-basis.

1. flex Shorthand Syntax

flex: <flex-grow> <flex-shrink> <flex-basis>;
  • flex-grow: Controls how much an item will grow relative to others.
  • flex-shrink: Controls how much an item will shrink when space is tight.
  • flex-basis: Defines the starting size of an item before growth or shrinking.

If you omit values, defaults are used: 

flex: 0 1 auto;

This means:

  • flex-grow: 0 → Item will not grow.
  • flex-shrink: 1 → Item can shrink if necessary.
  • flex-basis: auto → Size is based on the content or width/height set.

2. Sub-properties

flex-grow (default: 0)

Specifies how much a flex item will grow relative to others when extra space is available.

  • A value of 0 means the item won’t grow.
  • A value of 1 (or higher) allows the item to grow proportionally.

Example:

.item1 {
  flex-grow: 1; /* Will grow to fill extra space */
}
.item2 {
  flex-grow: 2; /* Will grow twice as much as item1 */
}

flex-shrink (default: 1)

Specifies how much a flex item will shrink relative to others when space is tight.

  • A value of 0 prevents shrinking.
  • A value of 1 (or higher) allows proportional shrinking.

Example:

.item1 {
  flex-shrink: 1; /* Will shrink if space is limited */
}
.item2 {
  flex-shrink: 0; /* Won't shrink */
}

flex-basis (default: auto)

Defines the initial size of the flex item before growing or shrinking.

  • Can be set in absolute units (px, %) or auto (content size).
  • Overrides width/height unless set to auto.

Example:

.item {
  flex-basis: 200px; /* Item starts at 200px wide */
}

3. Using the Shorthand

Example 1: Equal Growth, No Shrinking, Fixed Basis

.item {
  flex: 1 0 100px; /* Will grow, won't shrink, starts at 100px */
}

Example 2: Proportional Growth

.item1 {
  flex: 1; /* Equal growth */
}
.item2 {
  flex: 2; /* Grows twice as much as item1 */
}

Example 3: Preventing Growth and Shrinking

.item {
  flex: 0 0 150px; /* Fixed size of 150px, won't grow or shrink */
}

4. Practical Example

HTML:

<div class="container">
  <div class="item item1">Item 1</div>
  <div class="item item2">Item 2</div>
  <div class="item item3">Item 3</div>
</div>

CSS:

.container {
  display: flex;
}

.item1 {
  flex: 1; /* Takes equal space */
}
.item2 {
  flex: 2; /* Takes twice the space of item1 */
}
.item3 {
  flex: 1; /* Same size as item1 */
}

Result:

  • Item 2 will be twice as wide as Items 1 and 3.

Key Points

  • Use flex-grow for distributing extra space.
  • Use flex-shrink for managing shrinking behavior in tight spaces.
  • Use flex-basis to define the initial size of the item.
  • Combine all three with the flex shorthand for clean and powerful layouts.

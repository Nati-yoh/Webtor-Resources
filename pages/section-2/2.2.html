CSS can be included in HTML in three primary ways:

  1. Inline CSS: Adds styles directly to individual HTML elements using the style attribute. Useful for quick, specific changes but not ideal for larger projects.

    <p style="color: blue; font-size: 16px;">This is an inline-styled paragraph.</p>

  2. Internal CSS: Adds styles within the <style> tag inside the <head> section. This approach styles the entire HTML document but keeps the CSS separate from content structure.

    <head>
   <style>
      p {
         color: green;
         font-size: 18px;
      }
   </style>
</head>

  3. External CSS: Links to an external CSS file, typically named style.css, which contains all the styles for the website. This method is the best for large projects, as it keeps HTML and CSS separate and makes styling changes easier to manage across multiple pages.

    <head>
   <link rel="stylesheet" href="style.css">
</head>

Each method serves different needs, with external CSS being the preferred approach for organized, scalable projects.

To code in CSS, you select an HTML element and set properties to style it. Here’s a basic example using the color property to change text color:

CSS Syntax

CSS follows this structure:

selector {
    property: value;
}

Using the color Property

Let's apply the color property to make text appear in different colors.

/* This sets all <p> elements to have red text */
p {
    color: red;
}

/* This sets all <h1> elements to have blue text */
h1 {
    color: blue;
}

/* This sets elements with the class "highlight" to have green text */
.highlight {
    color: green;
}

/* This sets elements with the id "main-title" to have purple text */
#main-title {
    color: purple;
}

In HTML, these styles will change the colors of elements based on the selectors:

<h1 id="main-title">This is the Main Title</h1>
<p>This paragraph text will be red.</p>
<p class="highlight">This highlighted text will be green.</p>

Using the color property, you can control the text color for different elements on your webpage.

Css selectors

In CSS, selectors are used to target HTML elements and apply styles to them. Here’s a breakdown of the most common CSS selectors:

1. Universal Selector (*)

Selects all elements on the page.

* {
   color: black;
}

2. Type Selector (Element Selector)

Selects all elements of a specific type (like <p>, <h1>, etc.).

p {
   color: blue;
}
h1 {
   color: red;
}

3. Class Selector (.)

Selects all elements with a specific class. Use a period . followed by the class name.

.highlight {
   color: green;
}

HTML:

<p class="highlight">This text is green.</p>

4. ID Selector (#)

Selects an element with a specific ID. Use a hash # followed by the ID name.

#main-title {
   color: purple;
}

HTML:

<h1 id="main-title">This title is purple.</h1>

5. Descendant Selector (Space)

Selects elements that are nested inside other elements.

div p {
   color: orange;
}

HTML:

<div>
   <p>This text inside a div is orange.</p>
</div>

6. Child Selector (>)

Selects elements that are direct children of another element.

div > p {
   color: yellow;
}

HTML:

<div>
   <p>This paragraph is directly inside a div and is yellow.</p>
   <section>
      <p>This paragraph is inside a section, not directly inside the div.</p>
   </section>
</div>

7. Attribute Selector ([])

Selects elements based on an attribute or its value.

input[type="text"] {
   background-color: lightblue;
}

HTML:

<input type="text" value="Enter text here">

8. Pseudo-Classes (:)

Selects elements in a specific state, like when a user hovers over a link.

a:hover {
   color: red;
}

9. Pseudo-Elements (::)

Selects a specific part of an element, like the first letter or line.

p::first-letter {
   font-size: 2em;
   color: blue;
}

Examples in HTML and CSS:

HTML:

<div>
   <p class="highlight" id="special">This paragraph is styled with class and ID.</p>
   <p>This paragraph is just normal.</p>
   <a href="#">Hover over me</a>
</div>

CSS:

/* Universal selector */
* {
   font-family: Arial, sans-serif;
}

/* Class selector */
.highlight {
   font-weight: bold;
   color: green;
}

/* ID selector */
#special {
   color: blue;
}

/* Pseudo-class selector */
a:hover {
   color: red;
}

This covers the basic ways you can select elements in HTML using CSS. By combining different selectors and properties, you can create sophisticated and responsive designs.
